# Kougaku

Kougaku je stránka zajišťující všechny potřeby při vaší cestě sbírání hudby, seriálů či filmů. Její účel je jednoduché indexování všech populárních forem fyzických médií, které by jste chtěli sbírat, do jednoduché knihovny. Mezi podporované média patří gramofonové desky, CD, kazety, DVD, Blu-Ray a další.

## Funkce

Hlavní sekce aplikace je knihovna hudby kategorizovaná podle alba. Každé album má odkazy na streamovací služby + Discogs a Musicbrainz stránky. Rozhraní je příbuzné uživatelskému rozhraní aplikací na poslech hudby, přesto ale má extra funkce přizpůsobené k fyzickým médiím. Mezi ně patří například importování CUE/BIN párů do knihovny, automatický tagging hudebních souborů, integrace s [Discogs](https://discogs.com)/[Musicbrainz](https://musicbrainz.org), atd.
